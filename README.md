# User Management & Data Automation System

A full-stack web application for managing user records with secure CRUD operations, automated data generation, and RESTful API architecture. Built using Node.js, Express.js, MySQL, and EJS.

## 🚀 Features

* Create, Read, Update, and Delete (CRUD) user records
* Server-side rendered UI using EJS templates
* MySQL database integration for persistent storage
* RESTful API architecture
* Secure Update and Delete operations with credential verification
* Server-side input validation
* Automated database seeding using Faker.js
* Scalable testing with 100+ generated user records

## 🛠️ Tech Stack

* **Backend:** Node.js, Express.js
* **Database:** MySQL
* **Frontend:** EJS, HTML, CSS
* **Data Generation:** Faker.js
* **API Style:** RESTful APIs

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/user-management-system.git
```

2. Navigate to the project directory

```bash
cd user-management-system
```

3. Install dependencies

```bash
npm install
```

4. Configure MySQL database credentials.

5. Start the application

```bash
node app.js
```

6. Open your browser and visit:

```bash
http://localhost:8080
```
